# Student Performance Analyzer

## 📌 Description
This project is built using Python and SQLite to analyze student performance.

## 🚀 Features
- Add student data
- View records
- Calculate average
- Rank students
- Assign grades

## 🛠️ Technologies
- Python
- SQLite

## ▶️ How to Run
1. Install Python
2. Run:
   python student_analyzer.py
